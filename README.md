# Winnergy — Schulprojekt

Kurzanleitung zum Starten der lokalen Seite:

1. Öffne das Verzeichnis `winnergy-site`.
2. Öffne die Datei `index.html` in deinem Browser (Doppelklick oder `Strg+O`).

Optional: Lokaler HTTP-Server (empfohlen) mit Python:

```bash
# Python 3
python -m http.server 8000
# Dann Browser auf http://localhost:8000 öffnen
```

Hinweis: Die Website verwendet GSAP von CDN für Animationen. Keine weitere Installation nötig.
